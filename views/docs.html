How to use it

9jaworld can be used with any type of front end project that needs information of countries, states, local governments or any past leaders in Nigeria in the JSON format. you can use the below examples to check how it works.

Countries:

Get all countries

Route: "ninejaworld-api.onrender.com/countries"

Sort: "ninejaworld-api.onrender.com/countries?sort=name" This code sorts the countries by their name:

Sort: "ninejaworld-api.onrender.com/countries?sort=name&desc=true" by default the items are sorted in ascending order, to sort in descending order use the desc parameter:

Get a single countries

Get a country by it ID: "ninejaworld-api.onrender.com/countries/:countryID"

Search for countries

Search: You can search for countries by their name, continent, currency and/or capital: "ninejaworld-api.onrender.com/countries/search?name=niger"

Sort: This code searched for countries that the name includes "land", sorts them by their name: "ninejaworld-api.onrender.com/countries/search?name=land&sort=name"

Desc: by default the items are sorted in ascending order, to sort in descending order use the desc parameter: "ninejaworld-api.onrender.com/countries/search?name=land&sort=name&desc=true"
