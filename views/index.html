9jaworld API

Introduction

This API documentation describes the RESTful API for the 9jaworld API. The API allows developers to interact with and use the informations about different countries in the world. The api was designed to give certain details (such as population, continent, flag, currency etc) on all the countries all over the world, details (such as region, governors, senators, house of reps etc) of the 36 states in Nigeria and all the Local government areas in the country. There's also an endpoint to get the list of past Presidents and Governors in Nigeria with certain details about their government.

Getting Started

To get started with the API, you don't need to create an account or obtain an API key. All you need to do is send your request to the right Endpoints and you're good to go.

/countries 150+ countries

/states 36 states in Nigeria

/lgas 500+ local government areas

/presidents 16+ Nigeria Presidents

/governors 150+ Nigeria Governors
